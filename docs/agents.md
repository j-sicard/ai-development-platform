# Organisation des Agents IA

## Présentation

La plateforme utilise une équipe d'agents IA spécialisés représentant différents métiers d'une équipe professionnelle de développement logiciel.

Chaque agent possède un domaine d'expertise afin de couvrir les différentes phases d'un projet.

---

# Les agents

## 1. Product Owner / Business Analyst Senior

### Responsabilités

- Analyse des besoins métier
- Définition des objectifs fonctionnels
- Rédaction des user stories
- Priorisation des fonctionnalités

---

## 2. Research Analyst Senior

### Responsabilités

- Recherche technique
- Analyse des solutions existantes
- Comparaison des technologies
- Veille technologique

---

## 3. Architecte Logiciel Senior

### Responsabilités

- Conception de l'architecture
- Choix techniques
- Définition des composants
- Validation des approches

---

## 4. Backend Developer Senior

### Responsabilités

- Conception des API
- Développement de la logique métier
- Création des services backend
- Gestion des données

---

## 5. Frontend Developer Senior

### Responsabilités

- Conception de l'interface utilisateur
- Expérience utilisateur
- Développement des composants frontend

---

## 6. DevOps Engineer Senior

### Responsabilités

- Gestion Docker
- Déploiement
- Automatisation
- Gestion de l'environnement technique

---

## 7. QA Engineer Senior

### Responsabilités

- Stratégie de tests
- Tests fonctionnels
- Validation qualité

---

## 8. Security Engineer Senior

### Responsabilités

- Analyse des risques
- Sécurité applicative
- Application des bonnes pratiques de sécurité

---

## 9. Technical Writer / Documentation Engineer

### Responsabilités

- Documentation technique
- Guides utilisateurs
- Synthèse des travaux
